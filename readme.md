## 1.制作アプリのタイトル
- 血糖値管理アプリ

## 2.制作アプリの説明（40文字程度）
- 世の中にあるカロリー計算ではなく、無料でわかる範囲で簡易的に血糖値の上昇状況を確認するアプリ

## 3.工夫した点・こだわった点
- chart.jsを利用してグラフ化したところ。疑似的なので、正式なグラフではありません。
- 日本フードの無料検索APIを利用。

## 4.次回トライしたこと（または機能）
- 世の中で課題とされているものは、容易なものは簡単に無料でアプリ化されていた。
- figmaをもっと勉強する。
- firebaseをもっと勉強する。
